# Summarises a list of EOO or AOOgrid objects in a table

`list2table` wrapper that summarises a list of EOO or AOOgrid objects as
a table.

## Usage

``` r
list2table(l)
```

## Arguments

- l:

  a list of EOO or AOOgrid objects

## Value

a data.frame
