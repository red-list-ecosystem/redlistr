# Mangrove area

Area of mangroves in 2017

## Usage

``` r
a.2017
```

## Format

### `a.2017`

A data frame with 1 row and 3 columns:

- layer:

  name of the layer

- value:

  raster value from which area was computed

- area:

  area in km2

## Source

computed from mangrove.2017 using getArea
