# Mangrove area combined

Area of mangroves in 2000 and 2017

## Usage

``` r
a
```

## Format

### `a`

A data frame with 2 rows and 3 columns:

- layer:

  name of the layer

- value:

  raster value from which area was computed

- area:

  area in km2

## Source

computed from mangrove.2000 and mangrove.2017 using getArea
